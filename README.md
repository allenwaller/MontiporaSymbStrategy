# MontiporaSymbStrategy
Code and data for testing effects of symbiont population traits on coral bleaching recovery using the Hawaiian rice coral Montipora capitata hosting Cladocopium sp. or Durusdinium glynni. Data from summer 2019, Kane'ohe Bay, Hawai'i. Paper published in Symbiosis Jan 2023 (Allen-Waller, L., Barott, K.L.
